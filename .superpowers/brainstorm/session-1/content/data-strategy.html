Estrategia de Datos y Rendimiento

Queremos "todos los glaciares posibles", pero cada análisis requiere datos históricos y climáticos.

A

Pre-calculado (Más rápido)

Procesamos los datos de Copernicus para una lista extensa (ej. 500-1000 glaciares) y guardamos el riesgo base. El mapa carga instantáneamente.

B

En tiempo real (Más actual)

Calculamos todo al hacer clic. Permite seleccionar CUALQUIER punto, pero puede tardar 5-10s por las llamadas a APIs externas.

C

Híbrido (Recomendado)

Mapa de calor pre-calculado para visión global rápida. Al seleccionar un glaciar específico, se disparan los datos climáticos en vivo para la predicción final.
