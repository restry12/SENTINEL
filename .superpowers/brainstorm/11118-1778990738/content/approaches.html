A — Botón → Dropdown (Recomendado)

Un botón "TOP FOCOS" en el topbar. Al hacer clic abre un panel dropdown con input de búsqueda arriba y lista top 10 abajo. Simple y consistente con el resto del topbar.

Pros

  • No ocupa espacio cuando está cerrado
  • Familiar (patrón de dropdown estándar)
  • Fácil de implementar

Cons

  • Un clic extra para abrir

B — Command Palette (Modal)

Overlay centrado tipo Spotlight/VS Code. Más espacio para mostrar coordenadas y detalles. Se cierra con ESC o clic fuera.

Pros

  • Más espacio para info (coords, intensidad)
  • Fácil de buscar con teclado

Cons

  • Tapa el mapa
  • Más complejo de implementar

C — Side Drawer

Un drawer que se desliza desde la derecha al activarse. La lista queda siempre visible mientras navegas el mapa sin cerrarlo.

Pros

  • Lista siempre visible al mismo tiempo que el mapa

Cons

  • Achica el mapa
  • Más complejo (ya hay panel derecho)
